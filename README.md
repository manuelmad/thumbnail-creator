Creador de miniaturas (TikTok / YouTube)

Instrucciones rápidas:

- Abrir `index.html` en un navegador.
- Por defecto está seleccionada la plataforma TikTok (118x224).
- Cambiar a YouTube y elegir orientación para ver las dimensiones.
- Hacer clic en el recuadro o arrastrar una imagen para subirla.
- Escribir el título y ajustar el tamaño de la fuente; se verán en tiempo real.

Servidor local recomendado (desde la carpeta del proyecto):

```pwsh
python -m http.server 8000
```

Luego abrir http://localhost:8000 en el navegador.
